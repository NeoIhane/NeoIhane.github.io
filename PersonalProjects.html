Momentum Game Play tangible on multi-touch Table

Thesis - 2015

Tools: Unity, reacTivision, Visual studio (C#)

Technical: TUIO protocol, Frustrated Total Internal Reflex(FTIR), Physic, Bluetooth, Tactile

Info

This is solving problem momentum physic game by using tangible interface and touch on multitouch table. I put 2 sensors in tagible (switch for on/off and stick with joystrick sensor for control force in game), created physic game by using physic2D in Unity, created and write protocal for communication in between tangible and application via serialport. I build Multi-touch table for testing by using TUIO protocal, Frustrated Total Internal Reflex(FTIR) technique, and reacTivision (Program that process image from camera and send TUIO data).

Play piano with cybergrove. Orge3D, C/C++

Class Project

Tools: Orge3D, Visual studio (C/C++), Blender

Mechanic: CyberGrove, Microcontroller, Vibration motors, Led RGB

Info

Class project in human computer interface class (2 persons). Receive signal from cyberglove and joystick to control visual hand for playing piano. When position of finger touch on the key of piano it will generate sound of note with midi and send signal to microcontroller to drive vibration motors and show light on finger that touch. I did program and application parts and my friend did microcontroller and circuit parts.
